Buckets 0.67 - Fuzzy search!

Jul 18, 2022 • Matt

I’m excited to announce that Buckets Beta v0.67.0 is out now! It includes a few minor fixes, but two important new features I’ll detail below. Grab the beta version here or wait a week or two to get these improvements in the stable version.

Fuzzy search for categories¶

When categorizing transactions, you used to have to type in the bucket name exactly as it was written. Now, however, you can search more naturally for any part of the bucket name.

See below how “propan” matched both the “Natural gas/propane/oil” bucket as well as the “Personal Loan Payment” bucket.

This will be particularly helpful if you put Emojis in your bucket names. (Also, don’t tell anyone but the introduction of this fuzzy search dropdown is a precursor to the planned payee field.)

Colorblind friendly colors¶

We want Buckets to be available and useful to as many people as possible, yet in the latest community survey, you told us that the default colors are not colorblind friendly. So we fixed it!

You can now choose from 8 colorblind friendly hues in the default palette (highlighted in red) or choose any other color you want with the new color picker.

— Matt
